# UI 设计规范 (iOS 风格)

> 参考 iOS Human Interface Guidelines，保持品牌紫色调性

## 一、圆角规范

| 元素 | 圆角 | 说明 |
|------|------|------|
| 按钮 | 12px | 主要、次要按钮统一 |
| 小按钮 | 8px | size="sm" 的按钮 |
| 图标按钮 | 10px | IconButton |
| 卡片 | 16px | 内容卡片 |
| 列表项 | 12px | 列表中的卡片项 |
| 弹窗 | 16px | Dialog/Modal |
| 输入框 | 10px | Input/Textarea |
| 表格 | 12px | Table 容器 |
| Badge | 6px | 标签徽章 |

## 三、阴影规范

| 类型 | 阴影值 | 用途 |
|------|--------|------|
| 卡片默认 | 0 2px 12px rgba(0,0,0,0.08) | 内容卡片 |

| 弹窗 | 0 8px 32px rgba(0,0,0,0.16) | 模态框 |
| 下拉菜单 | 0 4px 16px rgba(0,0,0,0.12) | Menu/Dropdown |

## 六、交互规范

### 按钮

```
默认: 圆角12px + 内边距12px 24px
悬停: 背景加深 10%
按下: 背景加深 20% + 轻微缩放(0.98)
禁用: 透明度 50% + 禁止点击
```

### 输入框

```
默认: 边框 1px #e5e7eb
悬停: 边框 1px #d1d5db
聚焦: 边框 2px #7c3aed + 阴影
错误: 边框 2px #ef4444
```

### 卡片

```
默认: 圆角16px + 阴影
悬停: 上浮 4px + 阴影增强
点击: 轻微缩放(0.99)
```
